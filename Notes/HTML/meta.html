How should we order our attributes?

  1. class
  2. id, name
  3. data-*
  4. src, for, type, href, value
  5. title, alt
  6. aria-*, role

Block elements

  • paragraphs (p tag)
  • headers (h1, h2, etc tags)
  • lists (ul, ol, and li tags)
  • divs (div tag)

Inline elements

  • hyperlinks (a tag)
  • images (img tag)
  • form fields (input tag)
  • spans (span tag)
